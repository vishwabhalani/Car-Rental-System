# Car-Rental-System
The Car Rental System is a console-based, menu-driven Java program designed to simulate the basic operations of a real-world car rental service. The project focuses on implementing object-oriented programming concepts, user input handling, conditional logic, and loop-based menu navigation.

This program allows users to rent cars, return rented cars, and exit the application, while maintaining accurate customer and rental records. The program ensures data validation and logical checks to prevent incorrect operations, such as returning a car that was not rented by the customer.

✨Key Features:
-Menu-Driven System: Provides options to rent a car, return a car, or exit using a loop-based menu.
-Automatic Customer ID Generation: Generates and increments a unique customer ID for each new rental.
-Car Rental Process: Collects customer name, car ID, and rental days to process rentals.
-Rental Information Display: Shows complete rental details including price calculation.
-Confirmation Before Rental: Asks user confirmation (Y/N) before finalizing the rental.
-Car Return Validation: Ensures only the correct customer can return the rented car.
-Object-Oriented Design: Uses Java OOP concepts for clean and modular code structure.
